## üìñ About The Project

**DeepVerify** is a modern web application designed to help users identify whether an image is an authentic photograph or generated by artificial intelligence. In an era where AI-generated content is becoming increasingly sophisticated and widespread, DeepVerify aims to combat misinformation by providing a reliable verification tool.

### üéØ Key Features

- **üñºÔ∏è Image Upload** - Drag & drop or click to upload images (JPG, PNG, WEBP)
- **‚ö° Real-time Analysis** - Fast AI-powered image classification
- **üìä Confidence Score** - Get percentage-based confidence for each prediction
- **üé® Modern UI** - Sleek, responsive design with smooth animations
- **üîí Privacy First** - Images are processed in-memory and never stored
- **üì± Fully Responsive** - Works seamlessly on desktop and mobile devices

## üöÄ Getting Started

### Prerequisites

- **Node.js** >= 18.x
- **Python** >= 3.9
- **npm** >= 9.x or **yarn** >= 1.22.x

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Radityas-hub/DeepVerify.git
   cd DeepVerify
   ```

2. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

3. **Setup Backend**
   ```bash
   cd backend
   python -m venv venv
   venv\Scripts\activate  # Windows
   # source venv/bin/activate  # Mac/Linux
   pip install -r requirements.txt
   python app.py
   ```
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

### Build for Production

```bash
npm run build
npm run preview
```

## üìö Documentation

### How It Works

1. **Upload** - User uploads an image through drag & drop or file selection
2. **Processing** - Image is validated and preprocessed
3. **Analysis** - AI model analyzes the image for authenticity markers
4. **Result** - Classification result displayed with confidence score

### Supported Formats

| Format | Max Size |
|--------|----------|
| JPEG (.jpg, .jpeg) | 5MB |
| PNG (.png) | 5MB |
