## üìñ About The Project

**DeepVerify** is a web application that uses a trained Convolutional Neural Network (CNN) to detect whether an image is an authentic photograph or generated by artificial intelligence. Built to combat misinformation in the era of AI-generated content.

### üéØ Key Features

- **üß† Real AI Model** - Trained CNN model (`ai_vs_real_cnn.h5`) for image classification
- **üñºÔ∏è Image Upload** - Drag & drop or click to upload images (JPG, PNG, WEBP)
- **‚ö° Real-time Analysis** - Fast prediction with confidence score
- **üé® Modern UI** - Sleek, responsive design with GSAP animations
- **üîí Privacy First** - Images processed in-memory, never stored

## üöÄ Getting Started

### Prerequisites

- **Node.js** >= 18.x
- **Python** >= 3.9
- **npm** >= 9.x

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Radityas-hub/DeepVerify.git
   cd DeepVerify
   ```

2. **Setup Backend**
   ```bash
   cd backend
   
   # Create virtual environment (optional)
   python -m venv venv
   venv\Scripts\activate  # Windows
   # source venv/bin/activate  # Mac/Linux
   
   # Install dependencies
   pip install -r requirements.txt
   
   # Add your trained model (ai_vs_real_cnn.h5) to backend folder
   
   # Run server
   python app.py
   ```
   Backend will run at `http://localhost:5000`

3. **Setup Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
   Frontend will run at `http://localhost:5173`

4. **Open in browser**
   ```
   http://localhost:5173
   ```

## üß† AI Model

### Model Architecture
- **Type**: Convolutional Neural Network (CNN)
- **Input**: 224x224 RGB images
- **Output**: Binary classification (sigmoid)
- **File**: `ai_vs_real_cnn.h5`
